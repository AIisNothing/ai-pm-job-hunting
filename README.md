# AI时代下的产品经理怎么找工作

> 基于 6000+ 真实招聘数据的北京产品经理市场深度分析，帮你看清方向、避开误区、做对决策。

## 🎯 为什么做这件事

传统的找工作方式本质上是随机的——自己在招聘平台上翻岗位、凭感觉投简历，或者等猎头推荐。你很难回答"现在市场到底什么方向值钱""我的经验适合往哪走""高薪岗位到底多要求了什么"这些关键问题，因为你看到的永远只是局部信息。

AI 时代给了我们一个新的可能：用数据代替直觉。

AI 擅长处理大规模数据、发现人眼难以察觉的趋势、做多维度交叉分析。当我们把几千个真实岗位的薪资、方向、技能要求、公司特征汇聚在一起，很多原本模糊的判断就变得清晰了——哪些方向高薪岗位密集、哪些"热门方向"其实薪资平平、高薪 JD 和普通 JD 的能力差异到底在哪里。

这个项目就是一次实践：**用 AI 的数据处理和分析能力，辅助产品经理做更理性的求职决策。** 不是替代你的判断，而是给你的判断提供数据支撑。

## 这个项目是什么

一个用数据驱动的产品经理求职决策工具。不是教你写简历的鸡汤文，而是用真实招聘数据回答：

- 现在哪些方向高薪岗位最密集？
- 哪些方向看起来热门但薪资其实一般？
- 高薪 JD 和普通 JD 到底差在哪？
- 哪些"技能"只是标签，哪些是真正的能力信号？
- 5-10 年经验在哪些方向溢价最明显？
- 不同公司类型的薪资特征是什么？

## 核心发现

基于北京 6085 个产品经理岗位（3-10年·本科+·30K+）的分析：

| 发现 | 数据支撑 |
|------|---------|
| 多数方向中位数持平（37.5K），差异在高薪占比 | 高薪占比从 17% 到 38% 不等 |
| AI/大模型和策略产品高薪岗位最密集 | 高薪占比 36%+，P75 溢价 +5.6% |
| 高薪 JD 的核心差异不是工具技能 | 而是业务闭环、指标驱动、策略设计、AI落地、Owner能力 |
| 经验溢价在不同方向差异巨大 | 用户体验/C端 +33%，策略产品 +20% |
| 标签≠能力，高频标签不等于高薪原因 | 社交产品/用增产品只是方向标签 |

## 方法论

### 数据处理流程

```
原始招聘数据 → 薪资标准化 → 方向重新聚类 → 极端值剔除 → 多维交叉分析 → 报告生成
```

### 分析维度

- **方向分析**：17 个方向重新聚类（不依赖平台原始分类）
- **薪资分层**：中位数 / P75 / P90 / 高薪占比（P75+）
- **能力解构**：高薪 JD vs 普通 JD 的能力因子差异
- **技能分类**：方向标签 / 能力信号 / 小样本趋势词 / 无区分度标签
- **交叉验证**：经验×方向、行业×薪资、公司类型×高薪占比

### 审计规范

每次生成报告后执行：
- 数据溯源审计（独立复算核心指标）
- 数字一致性审计（同一指标跨章节一致）
- 口径一致性审计（不混用不同统计口径）
- 结论支撑审计（每个结论有数据依据）
- 小样本警告（<50 标注谨慎）

## 技术栈

- **数据存储**：SQLite + SQLAlchemy
- **数据分析**：pandas + numpy
- **报告生成**：Python 生成自包含 HTML（内嵌 ECharts）
- **图片导出**：Playwright 全页截图（2x Retina）
- **测试**：pytest + Hypothesis（属性测试）

## 如何使用

### 查看报告

直接打开 `report/V18_脱敏版.html`，任何浏览器都能看。

### 运行分析代码

```bash
# 安装依赖
pip install sqlalchemy numpy

# 生成报告（需要有数据库）
python analysis/generate_deep_report.py

# 生成脱敏公开版
python analysis/generate_public_report.py

# 生成最终交付文件（HTML + PNG）
python analysis/build_final_outputs.py
```

> 注意：运行分析代码需要有数据库文件（未包含在仓库中）。本仓库只提供代码框架和方法论，不提供原始数据。
